Stack de herramientas

  • Python es mi lenguaje predilecto. Los paradigmas de Programación Orientada a Objetos (OOP) y Programación Funcional me han dado las herramientas necesarias tanto para el análisis exploratorio de datos como para automatizar tareas.

  • Desde que he comenzado a trabajar para el Gobierno de la Ciudad Autónoma de Buenos Aires como analista de datos y luego como Coordinador Gral, Power Bi ha sido la manera más efectiva de comunicar a través de gráficos, la evidencia empírica que respalda la toma de decisiones.

  • No todas las bases de datos soportan conexiones SQL y muchas veces son simples hojas de cálculos. Es por esto, que me he capacitado en el uso de Excel y al día de hoy sigo utilizando esta poderosa herramienta para la limpieza y exploración de algunas bases dentro de la organización.

  • Git & GitHub son las herramientas de control de versiones para todos mis proyectos, desde mi comienzo en el campo del Desarrollo Web hasta mi inserción en el mundo de los datos. Actualmente, el repositorio de este website lo podrás encontrar en mi github

  • Jupyter Lab es mi entorno de desarrollo favorito junto a Visual Studio Code. Con Jupyter Lab puedo realizar mis informes y rápidamente tenerlos exportados en el formato que necesite (html, pdf, latex). Jupyter Lab y Git son la conbinación perfecta para el análisis de datos con Python.
